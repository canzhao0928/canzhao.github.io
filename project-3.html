Project Overview

This project is an inbox system designed for builders' use. It supports categorizing files by project and filtering projects by due date. When hovering over a single project, various operations such as deletion can be performed on that project.

1. Developed inbox web page with filter and edit features using React with Typescript, styling with Styled-component and testing with React Testing Library.
2. Deployed the project on AWS S3, continuing integration using GitHub workflows, ensuring efficient and reliable hosting for seamless access and scalability.
