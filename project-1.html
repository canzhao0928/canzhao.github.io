Project Overview

This project is an e-commerce website that sells electronic watches. Users can browse watches and add them to the shopping cart. By clicking on the image or title, they can enter the detailed product page and add items to the cart. It's also possible to remove items from the cart. The total value and quantity of items in the cart are automatically calculated and displayed. The system supports product management, allowing administrators to add, delete, modify, and query products after logging in.

1. Developed smartwatch display page and shopping cart using HTML/CSS, Bootstrap, and JavaScript.
2. Stored Products data in MySQL database by phpMyAdmin.
3. Implemented product management and login functions using PHP to update data from MySQL, allowing CRUD operations of product data and user authentication.
4. Successfully deployed the web application on Amazon EC2, ensuring reliable hosting and scalability for a seamless user experience.
